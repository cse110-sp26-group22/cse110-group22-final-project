# UI Hierarchy
## GameUI
- Header
  - The header is at the top and explains how to play the game.
- Plants
  - The plant in the middle grows as your score goes up, so as you get more answers correct, the plant grows.
- Input
  - The input at the bottom has ghost text that shows what you need to type, as well as the question that is shown that the user needs to answer.
- Scoring
  - The scoring system at the bottom is based on your CPM, your correctness, and how fast you beat the timer when you type. 
## Main Menu
- Title
  - Shows the title of our game, and explains how the game works.
- Start Button
  - There are two start buttons listed with the labels of the languages listed, either Python or JavaScript. Choosing one of these starts the game.
- Languages
  - When Python or JavaScript is picked, it will choose the questions of the respected language listed.
