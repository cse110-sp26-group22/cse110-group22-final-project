# AI Use For Prototyping the Frontend

Date: 5/19/2026   
AI Models: GPT Images 2.0, GitHub Copilot [ Auto ]

Prompts: 
1. can you make an image that contains 3 128 x 256 images of a vector graphic plant in its stages of growth (e.g. small, medium, large)? this is for a cozy webgame
2. make it a tomato plant. also make it look more vector-graphic

For quick prototyping, GPT Images 2.0 was used to generate three images of a tomato plant in its stages of growth (seedling $\to$ flower $\to$ fruit). The images were marked with the `-AI` suffix for easier tracking.

GitHub Copilot's code completions were used to accelerate frontend development. 
All code completions were manually reviewed and edited as they occurred.

AI Agents were not used in frontend development.
