# AI Use Logs

Document **any and all** use of AI here

## Rules:
If using AI, create a directory in /ai-use called /sprint-x-ai-log if it does not already exist and create a markdown file called your-name-ai-log.md. Include **at least** the following information: 
    - Date
    - What AI model is being used?
    - Prompts log
    - Summarize outputs/results used in final deliverable
